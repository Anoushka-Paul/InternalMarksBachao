# **App Name**: Deadline Dominator

## Core Features:

- Priority Deadlines Display: Display upcoming deadlines with subject name, description, optional marks, and due date, sorted by urgency.
- Deadline Editing: Allow users to edit existing deadline details (subject, description, marks, date).
- Completion Ticking: Enable users to mark deadlines as complete, removing them from the urgent list.
- History List: Maintain a list of deadlines categorized as 'due,' 'completed,' or 'overdue.'
- Exam Schedule Input: Provide a form for users to add exam schedules with subject, date, and time.
- Exam Schedule Display: Display the exam schedule in a clear, organized format.

## Style Guidelines:

- Primary color: Midnight blue (#191970) to promote a sense of calm focus.
- Background color: Dark gray (#333333) to provide a modern, dark theme aesthetic.
- Accent color: Steel blue (#4682B4) for interactive elements and to draw user attention.
- Body and headline font: 'Roboto', a sans-serif that balances modernity.
- Use simple, clear, white icons to represent actions and categories, such as a calendar for deadlines and a checkmark for completion.
- Employ a card-based layout to present deadlines and exam schedules in an organized manner. Prioritize the 'Priority Deadlines' section at the top.
- Use subtle transitions and animations for a smooth user experience when adding, editing, or completing deadlines.
